# Cross-Platform System Architecture Auditor

A lightweight, zero-dependency Python utility designed to audit system architecture, processor profiles, and operating system kernels.

## Project Purpose
This open-source tool maps out basic system metadata across different deployment environments. 

## Target Platforms
* Windows (Tested)
* Linux (Tested)
* macOS / Apple Silicon ARM64 (Development Pending - Requires testing on native M-series hardware to validate hardware abstraction layers)

## How to Run
Ensure you have Python 3 installed, then execute:
```bash
python app.py
```
